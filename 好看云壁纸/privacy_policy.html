好看云壁纸隐私政策

更新日期：2026年1月6日 | 生效日期：2026年1月6日

无限向量（北京）科技有限公司（以下简称“我们”或“好看云壁纸”）尊重并保护所有使用好看云壁纸产品和服务（以下简称“本服务”）的用户的隐私。当您开始使用本服务，即表示您已充分理解并同意我们按照本政策收集、使用、存储和保护您的个人信息。

我们深知隐私保护的责任重大，并致力于通过清晰、透明的说明，让您了解我们收集的信息类型、如何使用这些信息，以及您享有的控制权。请您仔细阅读本政策。

核心摘要：为了让您快速了解本政策重点，我们在此进行简要说明：

  • 服务定位：“好看云壁纸”是一款专注于提供壁纸浏览、下载、管理和创作分享的应用。
  • 信息收集：为了向您提供核心功能（如壁纸推荐、账户安全），我们会收集必要的设备信息、账号信息及使用行为。部分信息（如精确位置、相机）仅在您授权后收集。
  • 信息使用：我们收集的信息主要用于提供、优化和保障我们的壁纸服务，向您展示更符合您兴趣的壁纸内容和广告。
  • 您的权利：您可以通过应用内设置管理您的隐私偏好，并拥有访问、更正、删除个人信息及注销账户的权利。

一、我们收集哪些类型的信息？

我们收集的信息取决于您如何使用“好看云壁纸”，主要包括以下几类：

1. 您主动提供的信息

  • 账号信息：当您注册、登录时，我们可能会收集您的手机号码（用于一键登录或验证），以及您设置的昵称、头像、密码。
  • 壁纸内容：当您上传、发布或收藏壁纸时，我们会收集您上传的图片/视频、您添加的描述或标签。
  • 联系方式：当您联系我们或参与活动时，可能会提供您的电子邮箱地址或其他联系方式。

2. 在使用服务过程中自动收集的信息

  • 设备信息：为保障服务基本运行与安全，我们会收集您的设备型号、操作系统版本、唯一设备标识符（如Android ID）、IP地址、网络类型等信息。
  • 使用行为与日志信息：您浏览、搜索、下载、收藏壁纸的类型和频率，应用功能的使用情况（如自动换壁纸），以及应用崩溃日志。
  • 位置信息（需授权）：当您使用基于位置的壁纸推荐或希望在发布内容中添加位置标签时，在获得您的明确授权后，我们会收集精确或粗略的位置信息。
  • 相册/相机信息（需授权）：当您选择从相册上传壁纸或使用相机拍摄新壁纸时，在获得您的授权后，我们会访问您的相册或相机。

3. 来自第三方的信息

  • 当您选择使用微信、QQ等第三方账号登录时，我们会从该第三方处获取您授权共享的账号公开信息（如头像、昵称）。

4. 关于剪切板（仅限特定场景）

在您执行明确操作时（例如通过分享口令或链接接收他人分享的壁纸），我们可能会访问您的剪切板，以读取其中与此操作直接相关的信息（如分享码），用于实现跳转或分享功能。我们不会在后台持续读取或收集剪切板中的其他无关信息。

5. 第三方 SDK 收集的信息

为了提供推送服务、登录分享、统计分析、广告投放等功能，我们的应用中集成了第三方软件开发工具包（SDK）。这些 SDK 可能会根据其功能需求收集相关信息，具体请查阅本政策附录中的 《好看云壁纸接入的第三方 SDK 列表》。

二、我们如何使用这些信息？

我们将基于以下目的，在必要范围内使用您的信息：

  • 向您提供核心壁纸服务：例如，管理您的账户、展示和推荐壁纸、处理您的下载和收藏请求。
  • 优化和个性化服务：分析您的使用习惯，为您推荐更符合您口味的壁纸类别，并在不同设备上提供一致的体验。
  • 保障服务安全与维护：用于身份验证、安全防范、反欺诈、排查崩溃及故障，以保护您和我们的合法权益。
  • 与您沟通：向您发送服务通知、功能更新或回复您的咨询。
  • 广告与商业化：向您展示与您兴趣更相关的广告，并评估广告效果。我们绝不会向任何人出售您的个人信息。
  • 法律合规与义务：遵守适用的法律法规、响应司法机关要求或保护他人重要权益。

三、我们如何共享、转让、公开披露您的信息？

1. 共享

我们不会与第三方公司、组织或个人共享您的个人信息，但以下情况除外：

  • 获得您的明确同意后。
  • 与授权合作伙伴共享：仅为实现本政策中声明的目的，我们的供应商、服务提供商（如提供推送、数据存储、支付处理服务的公司）可能会在必要的范围内接触您的信息。我们要求他们遵守严格的保密义务。
  • SDK合作伙伴：如附录所列，为实现特定功能，SDK提供商会依据其隐私政策处理相关信息。
  • 基于法律要求：根据法律法规规定、诉讼争议解决需要，或应行政、司法等有权机关的要求。

2. 转让

我们不会将您的个人信息转让给任何公司、组织或个人，但在涉及合并、收购或资产转让时，我们会在要求新的持有您个人信息的公司、组织继续受本政策的约束，否则我们将要求该公司、组织重新向您征求授权同意。

3. 公开披露

我们仅会在获得您明确同意，或基于法律要求及保护公共利益的前提下，公开披露您的个人信息。

4. 您主动的公开与分享

请注意，您通过本服务主动公开分享的壁纸作品、个人资料信息（如昵称、头像）可能会被其他用户或公众看到，请谨慎考虑您分享的内容。

四、您的权利

您对自己的个人信息享有以下权利：

1. 访问与更正

您可以在“我的”-“设置”-“编辑资料”中访问和修改您的头像、昵称等个人资料。

2. 删除

您可以在“我的收藏”、“浏览历史”等板块中删除您的相关信息。您也可以通过注销账号来删除您的全部个人信息。

3. 注销账号

您有权随时注销您的“好看云壁纸”账号。

  • 注销路径：【个人中心】→【设置】→【账号管理】→【注销账号】。
  • 重要后果：账号注销是不可逆的操作。一旦注销，您的账号信息、个人资料、所有壁纸收藏、发布记录等数据将被永久删除且无法恢复，请您谨慎操作。

4. 撤回授权

您可以通过设备或应用内的权限设置，随时关闭相机、相册、位置等系统权限，以撤回对信息收集的授权。撤回授权后，我们将不再处理对应的个人信息，但不会影响此前基于您的授权已进行的处理。

5. 投诉与响应

如果您对您的个人信息处理有任何疑问或投诉，您可以通过本政策末尾的联系方式找到我们，我们将在15个工作日内予以回复。

五、我们如何保护您的信息

我们采用符合业界标准的安全防护措施（包括技术和管理措施）来防止您的信息遭到未经授权的访问、使用、修改、损坏或丢失。我们会定期对员工进行隐私保护培训，并仅允许有必要知晓信息的员工访问您的信息。

尽管我们已经尽力保护您的个人信息，但没有任何安全措施是完美的，我们无法做出绝对的安全保证。

六、本政策的更新

我们可能会适时修订本政策。当条款发生重大变更时，我们会在应用内通过弹窗、站内信等显著方式通知您，并更新本政策顶部的“更新日期”。请您定期查阅本政策。

如果您在更新后继续使用“好看云壁纸”服务，即表示您同意接受修订后的政策。
