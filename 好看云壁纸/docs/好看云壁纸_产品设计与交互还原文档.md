## 1. 全局设计规范

### 1.1 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| **主色 accent** | `#6366f1` | Indigo-500，按钮、链接、选中态 |
| **主色渐变** | `#6366f1 → #8b5cf6` | 权限弹窗 Icon、启动页 Logo 渐变 |
| **页面背景** | `#fdfdfe` | 近白色，全局统一 |
| **深色文字** | `#0f172a` (slate-950) / `#1e293b` (slate-800) | 标题 |
| **正文文字** | `#334155` (slate-700) | 正文 |
| **辅助文字** | `#64748b` (slate-500) | 描述、次要信息 |
| **弱化文字** | `#94a3b8` (slate-400) | 占位符、弱化 |
| **分割线/边框** | `#e2e8f0` (slate-200) | 卡片、输入框、分隔线 |
| **卡片面板** | `#ffffff` | 白色卡片/面板底色 |
| **浅灰背景** | `#f1f5f9` (slate-100) | 标签未选中、拒绝按钮 |
| **深色按钮** | `#1e293b` (slate-800) | CTA 主按钮（如订阅、登录） |
| **危险色** | `#ef4444` (red-500) | 删除；`#FF3B30` iOS 拒绝按钮 |
| **成功色** | `#10b981` (emerald-500) | 轮播中 badge |
| **遮罩** | `rgba(0,0,0,0.45)` | 弹窗背景遮罩 |
| **启动页遮罩** | `rgba(15,23,42,0.35)` + `blur(12px)` | 隐私弹窗遮罩 |

### 1.2 字体

| 属性 | 值 |
|------|-----|
| **字体族** | `Noto Sans SC`，fallback `sans-serif` |
| **字重** | 300(Light) / 400(Regular) / 500(Medium) / 700(Bold) / **900(Black)** |
| **抗锯齿** | `-webkit-font-smoothing: antialiased` |

> **核心原则**：标题和重要 CTA 统一使用 **font-weight: 900 (Black)**；正文 400-700。

### 1.3 圆角体系

| 场景 | 圆角值 | 说明 |
|------|--------|------|
| 大卡片（产品卡、Hero 区） | `2.5rem`（40px） | 含壁纸产品卡、相册 Hero |
| 标准卡片 | `2rem`（32px） | 设置分组、提示卡、隐私弹窗 |
| 按钮 | `1rem`（16px）/ `1.25rem`（20px）/ `2xl`（16px） | 各类按钮 |
| 标签（Tab/Tag） | `100px`（全圆角胶囊） | 分类 Tab、教程 Tab |
| 导航栏 | `2.5rem`（40px） | 底部 Nav |
| 图片缩略图 | `1.25rem`（20px） | 照片网格 |
| 头像 | `50%`（全圆） | 个人中心头像 |
| 输入框 | `1rem`（16px） | 登录/修改输入框 |

### 1.4 阴影体系

| 场景 | 阴影 |
|------|------|
| 大卡片 | `0 20px 40px -15px rgba(0,0,0,0.12)` |
| 标准卡片 | `0 shadow-sm`（Tailwind 默认 sm） |
| 底部导航栏 | `0 20px 50px rgba(0,0,0,0.15)` |
| 隐私弹窗 | `0 24px 48px rgba(0,0,0,0.08)` |

### 1.5 间距基准

- 页面水平内边距：`24px`（px-6）
- 卡片内边距：`16px` ~ `24px`
- Header 大标题区：`px: 32px, pt: 64px, pb: 24px`（含 safe-area）
- 组件间距常用：`6px / 8px / 10px / 12px / 16px / 24px`
- 底部预留：`pb-40`（160px）给导航栏让出空间

### 1.6 毛玻璃效果

## 2. 启动页

### 2.1 视觉

| 属性 | 值 |
|------|-----|
| 背景 | `#fdfdfe`（浅底，与首页一致） |
| Logo 尺寸 | `88 × 88 px` |
| Logo 圆角 | `22px` |
| Logo 来源 | **iOS**: `logo-ios.png` / **Android**: `logo-android.png`（CDN） |
| 标题 | "好看云壁纸"，`1.75rem`，font-weight 900，颜色 `#0f172a` |
| 副标题 | "展现精彩，看到爱"，`0.75rem`，颜色 `#64748b`，margin-top 6px |
| 布局 | 垂直居中：Logo → 标题 → 副标题 |

### 2.2 动画

| 元素 | 动画 | 参数 |
|------|------|------|
| Logo | 缩放淡入 | `0.5s cubic-bezier(0.16,1,0.3,1)`，from: opacity 0 + scale(0.85) |
| 标题 | 上移淡入 | `0.45s`，delay `0.15s`，from: opacity 0 + translateY(10px) |
| 副标题 | 上移淡入 | `0.45s`，delay `0.25s`，同上 |

### 2.3 消失逻辑

- 启动页显示 **1.5 秒**后自动消失
- 消失动画：`opacity: 0`，duration `0.5s ease-out`
- 若首次使用（未同意隐私），启动页 1.5 秒后保持（不消失），同时展示隐私弹窗
- 兜底：8 秒后无论如何进入主应用

### 2.4 平台 Logo 区分

| 平台 | Logo |
|------|------|
| iOS (iPhone/iPad/iPod) | `https://hk-h5-cdn.levect.com/html/xiefeng/CloudWallpaper/logo-ios.png` |
| Android 及其他 | `https://hk-h5-cdn.levect.com/html/xiefeng/CloudWallpaper/logo-android.png` |

> 通过 User-Agent 检测 `/iPad|iPhone|iPod/` 判断 iOS。

## 3. 隐私合规弹窗

### 3.1 展示时机

- 首次打开 App 且未同意过隐私协议时
- 在启动页 1.5 秒延迟后弹出（启动页不消失，隐私弹窗叠加在启动页之上）
- 同意后 `localStorage` 记录 `privacy_consent = '1'`

### 3.2 视觉

| 属性 | 值 |
|------|-----|
| 遮罩 | `rgba(15,23,42,0.35)` + `blur(12px)` |
| 卡片背景 | `#fdfdfe` |
| 卡片圆角 | `2rem`（32px） |
| 卡片最大宽度 | `22rem`（352px） |
| 卡片内边距 | `28px 24px` |
| 卡片阴影 | `0 24px 48px rgba(0,0,0,0.08)` |
| 卡片边框 | `1px solid rgba(226,232,240,0.9)` |
| 标题 | "隐私与条款"，`1.25rem`，font-weight 900，颜色 `#0f172a` |
| 描述 | `0.8125rem`（13px），颜色 `#64748b`，行高 1.55 |
| 链接 | `#6366f1`，font-weight 700，无下划线 |
| 链接内容 | `《用户协议》` → `https://92-cn.levect.com/92china/user_agreement` |
| | `《隐私政策》` → `https://92-cn.levect.com/92china/privacy_policy` |

### 3.3 按钮

| 按钮 | 样式 |
|------|------|
| **不同意** | 背景 `#f1f5f9`，文字 `#64748b` |
| **同意** | 背景 `#6366f1`，文字 `#fff` |
| 通用 | flex: 1, padding `14px 20px`, 圆角 `1rem`, font-weight 800, font-size `0.9375rem` |
| 按下 | `transform: scale(0.98)` |

### 3.4 交互

- **同意** → 记录 localStorage → 启动页消失 → 进入主应用
- **不同意** → 直接进入主应用（不记录，下次仍弹出）

### 3.5 入场动画

卡片：`0.35s cubic-bezier(0.16,1,0.3,1)`，from: `opacity 0, scale(0.96), translateY(16px)`
