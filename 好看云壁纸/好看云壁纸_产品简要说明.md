# 好看云壁纸 APP — 产品简要说明

> **文档版本**：v1.0  
> **日期**：2026-03-10  
> **目标读者**：研发团队  
> **目的**：简明扼要地说清楚产品是什么、有哪些功能、核心流程和平台差异，便于研发快速理解全貌。

## 一、产品定位

**好看云壁纸** 是一款手机壁纸订阅与锁屏照片轮播工具，支持 iOS 和 Android 双平台。

核心 slogan：**"展现精彩，看到爱"**

用户可以通过「订阅壁纸专题」和「上传个人照片」两种方式，实现锁屏壁纸的自动轮播更换。

---

## 二、核心功能模块（3 个 Tab）

| Tab | 名称 | 核心能力 |
|-----|------|----------|
| 1 | **壁纸订阅** | 浏览壁纸专题 → 按分类筛选（热门推荐、旅行日记、文化艺术、知名IP 等 9 类）→ 查看详情 → 免费订阅 → 自动推送到锁屏 |
| 2 | **我的相册** | 上传个人照片 → 管理照片（增/删/多选）→ 照片自动轮播到锁屏 |
| 3 | **个人中心** | 登录/注册 → 管理个人信息（头像/昵称）→ 功能设置 → 查看订阅 → 账号管理 |

## 三、关键业务流程

### 1. 首次启动流程

```
启动页 → 隐私合规弹窗（首次）→ 同意进入主页 / 不同意也进入（下次再弹）
非首次 → 启动页 → 直接进入主页
```

### 2. 壁纸订阅流程

```
浏览壁纸卡片 → 点击进入详情 → 点击"立即订阅"（免费）→ 订阅成功 → 触发锁屏壁纸设置引导
```

### 3. 照片上传流程

```
点击"上传照片" → 系统权限弹窗（首次）→ 授权 → 打开相册选择器（多选）→ 照片入库 → 触发锁屏壁纸设置引导
```

### 4. 锁屏壁纸设置引导（核心差异）

订阅成功 / 照片上传完成后，均会弹出锁屏壁纸设置引导弹窗，按平台走不同流程：

#### iOS 流程（基于快捷指令）

| 步骤 | 说明 |
|------|------|
| 1. 检测快捷指令 | 检查用户是否已添加「好看云壁纸」快捷指令 |
| 2a. 已添加 → 执行设置 | 调用快捷指令 → 显示"正在设置壁纸…" → **等待快捷指令执行完成回调** → 提示"设置成功" → 关闭弹窗 |
| 2b. 未添加 → 引导安装 | 弹窗引导用户跳转 iCloud 添加快捷指令 → 添加成功后返回 → 执行设置（同 2a） |

> **iOS 重要机制**：快捷指令**每次执行只能设置一张壁纸**。如需自动轮播，用户需在「快捷指令 App → 自动化」中配置定时触发（教程/玩法中心有详细步骤引导）。

#### Android 流程（基于动态壁纸服务）

| 步骤 | 说明 |
|------|------|
| 1. 检测服务状态 | 检查动态壁纸服务是否已开启 |
| 2a. 已开启 → 直接设置 | 显示"正在设置壁纸…" → **等待壁纸实际设置完成** → 提示"设置成功" → 关闭弹窗 |
| 2b. 未开启 → 引导开启 | 引导点击"去设置" → 跳转系统动态壁纸设置页 → **返回后检测服务实际开启状态** → 设置完成 → 关闭弹窗 |

> **Android 重要机制**：动态壁纸服务开启后，**直接支持多张壁纸自动轮播**，无需额外配置。

> **研发注意**：H5 原型中的定时器（2s / 2.5s）仅为演示用，实际开发需对接真实的系统回调/状态检测，按执行结果决定成功或失败提示。

### 5. 壁纸轮播能力对比

| 能力 | iOS | Android |
|------|-----|---------|
| 单次设置壁纸 | 执行一次快捷指令 = 设置一张 | 由动态壁纸服务自动完成 |
| 自动轮播 | 需用户配置「快捷指令自动化」（定时/条件触发） | 开启服务即生效，自动多张轮播 |
| 切换间隔设置 | 在快捷指令自动化中配置触发频率 | APP 内设置项可直接调整 |
| 适用范围 | 订阅壁纸 + 上传照片均通过快捷指令 | 订阅壁纸 + 上传照片均通过动态壁纸服务 |

### 6. 订阅壁纸更新节奏

- 每个订阅专题**每天推送 1 张**新壁纸
- 历史壁纸**不保留**在轮播池中，次日被新图替换
- 当前版本无收藏功能

## 四、主要页面与弹窗清单

| 页面/弹窗 | 说明 |
|-----------|------|
| 启动页 | Logo + 品牌名，区分 iOS/Android Logo |
| 隐私合规弹窗 | 首次打开弹出，含《用户协议》《隐私政策》链接 |
| 壁纸列表页 | 分类标签栏 + 壁纸产品卡片列表 |
| 壁纸详情页 | 封面图 + 壁纸预览（横向滚动）+ 订阅权益 + 用户评价 + 订阅/取消按钮 |
| 锁屏全屏预览 | 模拟真实锁屏效果，支持左右滑动切换，可切换"原图/锁屏效果" |
| 相册页 | 照片网格（3 列）+ 管理模式（多选删除）+ 空态提示 |
| 教程模态框（玩法中心） | 分 iOS/Android 两个 Tab，时间轴步骤引导 |
| 登录弹窗 | 手机号 + 验证码/密码，支持切换登录模式 |
| 相册权限弹窗 | iOS 为底部 Action Sheet 风格，Android 为 Material 居中对话框 |
| 相册选择器 | 4 列网格，支持多选，选中有视觉反馈 |
| 锁屏壁纸设置引导 | 订阅成功 / 照片上传后弹出，按平台引导不同设置流程 |
| 切换间隔弹窗 | 仅 Android，设置壁纸自动切换的时间间隔 |
| 我的订阅列表 | 底部弹窗，展示已订阅的专题列表 |
| 修改昵称/头像 | 底部弹窗 |
| 账号管理 | 居中弹窗，含账号注销等操作 |
| 评价壁纸 | 底部弹窗，星级 + 文字评价 |

## 五、平台差异要点

| 差异项 | iOS | Android |
|--------|-----|---------|
| 启动 Logo | `logo-ios.png` | `logo-android.png` |
| 权限弹窗样式 | 底部 Action Sheet（iOS 系统风格） | 居中 Material Dialog |
| 教程默认 Tab | iOS 快捷指令教程 | Android 动态壁纸教程 |
| 壁纸设置机制 | 快捷指令（单次设一张，自动化实现轮播） | 动态壁纸服务（开启即轮播） |
| 锁屏自动切换开关 | 不显示 | 显示 |
| 切换间隔设置 | 不显示 | 显示 |
| 二级页面导航栏 | 44px 高，返回键紫色 | 56px 高，返回键深色 |

## 六、用户体系

- **游客态**：可浏览壁纸、订阅，不可同步数据
- **登录态**：手机号 + 验证码/密码登录，支持修改头像/昵称、账号管理、账号注销
- 当前版本：**v2.6.0**
