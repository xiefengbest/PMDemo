# 首页精选订阅交互逻辑

## 1. 展示位置与布局

- **位置**：首页分类内容区域的**最前面**（在组图壁纸和创作者卡片之前）
- **标题**：精选订阅
- **布局方式**：横向滚动，卡片可左右滑动浏览
- **卡片尺寸**：宽度 `w-72`（288px），宽高比 `3:2`（横向卡片）

## 2. 卡片内容结构

每个订阅卡片包含以下元素：

| 层级 | 内容 | 样式 |
|------|------|------|
| 背景 | 封面图 | 使用产品的 `cover` 字段，全屏铺满 |
| 遮罩 | 渐变层 | 从底部黑色80%透明度渐变到顶部透明 |
| 底部信息区 | 产品名称 | 加粗白色文字，截断显示 |
| 底部信息区 | 副标题 | 较小白色文字，截断显示 |
| 底部信息区 | 更新频率标签 | 半透明白色背景 + 白色文字 |
| 底部信息区 | 订阅按钮 | 根据订阅状态动态变化 |

## 3. 交互行为

### 3.1 点击卡片区域

- **触发条件**：点击卡片任意位置（除按钮外）
- **行为**：调用 `openDetail(productId)` 打开订阅详情模态框
- **详情页内容**：
  - 封面大图
  - 产品名称、描述
  - 全部壁纸预览网格
  - 订阅权益模块
  - 用户评价
  - 底部操作栏

### 3.2 点击订阅按钮

按钮文案和行为根据订阅状态动态变化：

#### 未订阅状态

| 属性 | 值 |
|------|------|
| 按钮文字 | 立即订阅 |
| 按钮样式 | 白色背景 + 深色文字 |
| 点击行为 | 1. `event.stopPropagation()` 阻止冒泡<br>2. 调用 `toggleSubB(this, productId)` 执行订阅<br>3. 显示加载动画："正在同步壁纸数据…"<br>4. 预加载壁纸图片<br>5. 显示成功提示："订阅成功！壁纸已准备就绪"<br>6. 如果开启了推送通知，显示锁屏引导模态框 |

#### 已订阅状态

| 属性 | 值 |
|------|------|
| 按钮文字 | 已订阅 |
| 按钮样式 | 半透明白色背景 + 白色文字 |
| 点击行为 | `event.stopPropagation()` 仅阻止冒泡，**不执行任何操作** |

## 4. 数据来源

- 从 `CAT_PRODUCTS` 数组中筛选 `count === undefined` 的产品（即订阅型壁纸，非组图壁纸）
- 根据当前分类（`cat` 参数）过滤显示

## 5. 视觉特点

- 卡片圆角：`rounded-2xl`
- 阴影：`shadow-sm` + 边框 `border-slate-200`
- 横向滚动容器隐藏滚动条（`scrollbar-hide`）
- 按钮点击有缩放动画：`active:scale-95`

## 6. 与"我的合辑"的区别

| 特性 | 首页精选订阅 | 我的合辑 |
|------|--------------|----------|
| 卡片方向 | 横向 3:2 卡片 | 竖向 banner 样式（也是 3:2 比例） |
| 排列方式 | 横向滚动 | 竖向列表排列 |
| 内容范围 | 所有订阅型产品 | 仅用户已订阅的产品 |

## 7. 相关代码位置

- **渲染函数**：`renderCatProducts(cat)` - [index.html](file:///Users/haokan/Documents/code/AndroidStudioProjects/PMDemo/好看云壁纸/index.html#L2873-L2907)
- **详情打开**：`openDetail(productId)` 
- **订阅操作**：`toggleSubB(btn, productId)`
- **详情栏更新**：`updateDetailBar(productId)` - [index.html](file:///Users/haokan/Documents/code/AndroidStudioProjects/PMDemo/好看云壁纸/index.html#L3879-L3900)

# 首页组图合辑交互逻辑

## 1. 展示位置与布局

- **位置**：首页分类内容区域，在"精选订阅"之后
- **布局方式**：竖向列表，每个组图合辑独立区块
- **渲染函数**：`renderCatCard(p)` - [index.html](file:///Users/haokan/Documents/code/AndroidStudioProjects/PMDemo/好看云壁纸/index.html#L2778-L2812)

## 2. 卡片内容结构

每个组图合辑卡片包含两个部分：

### 2.1 头部信息区

| 元素 | 内容 | 样式 |
|------|------|------|
| 产品名称 | 合辑名称 | `text-base font-black text-slate-800` |
| 数量标签 | 如"9张壁纸" | 灰色背景圆角标签 |
| 副标题 | 合辑描述 | `text-xs text-slate-500` 截断显示 |
| 操作按钮 | 设为壁纸/正在使用 | 根据设置状态动态变化 |

### 2.2 壁纸预览区

- **布局方式**：横向滚动，隐藏滚动条
- **卡片尺寸**：宽度 `120px`，宽高比 `9:19.5`（竖长型）
- **卡片样式**：圆角 `rounded-2xl`，灰色背景 `bg-slate-100`
- **图片加载**：懒加载 `loading="lazy"`
- **数据来源**：使用产品的 `previews` 数组，如无则使用 `PREVIEW_POOL_B` 前6张

## 3. 交互行为

### 3.1 点击壁纸卡片

- **触发条件**：点击任意壁纸预览卡片
- **行为**：调用 `openDetail(productId)` 打开**组图详情模态框**
- **详情页内容**：
  - 封面大图
  - 产品名称、描述
  - 全部壁纸预览网格（9:19.5 比例）
  - **隐藏订阅权益模块**（组图壁纸特有）
  - 用户评价
  - 底部操作栏（设为壁纸/正在使用）

### 3.2 点击操作按钮

按钮文案和行为根据设置状态动态变化：

#### 未设置状态

| 属性 | 值 |
|------|------|
| 按钮文字 | 设为壁纸 |
| 按钮样式 | 深色背景 `bg-slate-900` + 白色文字 |
| 点击行为 | 1. `event.stopPropagation()` 阻止冒泡<br>2. 调用 `toggleSubB(this, productId)` 执行设置<br>3. 显示加载动画："正在准备壁纸…"<br>4. 预加载壁纸图片<br>5. 显示成功提示："设置成功！壁纸已准备就绪"<br>6. **不显示**推送通知和锁屏引导（组图壁纸特有） |

#### 已设置状态

| 属性 | 值 |
|------|------|
| 按钮文字 | 正在使用 |
| 按钮样式 | 灰色背景 `bg-slate-200` + 灰色文字 |
| 点击行为 | `event.stopPropagation()` 仅阻止冒泡，**不执行任何操作** |

## 4. 数据来源

- 从 `CAT_PRODUCTS` 数组中筛选 `count !== undefined` 的产品（即组图壁纸）
- 根据当前分类（`cat` 参数）过滤显示
- 组图壁纸特征：
  - 有 `count` 属性（如"9张壁纸"）
  - 有 `previews` 数组（包含多张壁纸URL）
  - 有 `tushuo` 属性（图说文案）

## 5. 视觉特点

- 头部与预览区间距：`mb-3`
- 横向滚动容器：`overflow-x-auto scrollbar-hide`
- 滚动容器左右扩展：`-mx-4 px-4`（出血布局）
- 壁纸卡片间距：`gap-2.5`
- 按钮点击缩放动画：`active:scale-95`

## 6. 与"精选订阅"的区别

| 特性 | 组图合辑 | 精选订阅 |
|------|----------|----------|
| 卡片方向 | 竖向 9:19.5 卡片 | 横向 3:2 卡片 |
| 排列方式 | 竖向列表，每个合辑独立区块 | 横向滚动 |
| 预览展示 | 横向滑动展示多张壁纸 | 单张封面图 |
| 按钮文案 | 设为壁纸/正在使用 | 立即订阅/已订阅 |
| 订阅权益 | 隐藏 | 显示 |
| 推送通知 | 不显示 | 显示（如开启） |
| 锁屏引导 | 不显示 | 显示 |

## 7. 特殊产品：画境集 (id=102)

画境集是组图壁纸的特殊案例，具有以下特点：
- 使用订阅逻辑但文案不同
- 按钮：设为壁纸 → 正在使用 → 停止使用
- 隐藏订阅权益模块
- 不显示壁纸更新提醒
