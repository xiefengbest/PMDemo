## 3. 换账号迁移

Cursor 换号后云端 Memories 和 User Rules **不会**自动跟随，需要手动迁移。

### 换号前（旧账号仍可登录）

1. **User Rules** — 设置 → Rules → User Rules，全选复制 → 粘贴到 `USER_RULES.md`。
2. **Memories** — 设置 → Memories，逐条复制 → 粘贴到上方 [记忆条目](#1-记忆条目)。
3. **其他偏好** — 填写上方 [个人偏好](#2-个人偏好)（不写密钥）。
4. **确认提交** — 确保 `用户上下文汇总.md`、`USER_RULES.md`、`.cursor/rules/user-context-from-repo.mdc` 已保存。

### 换号后（新账号）

1. 登录新号，确认订阅与额度。
2. 将 `USER_RULES.md` 全文粘贴到新账号 User Rules。
3. 打开本仓库 — 项目规则自动加载，助手会参考本目录内容。
4. 将记忆条目在新账号 Memories 里重新添加（如支持）。
5. MCP / API Key 在新号重新配置。

### 换号后仍可能丢失

- 旧账号云端聊天记录、团队规则、未写入本文件的条目。
