一、我们如何收集和使用您的个人信息

1.1 我们收集的信息类型

（1）账号信息

当您注册或使用我们的服务时，我们会收集：

  • 手机号码：用于账号注册、登录验证和安全保护
  • 昵称：用于个人资料展示
  • 头像：用于个人资料展示
  • 第三方账号信息：如果您选择通过微信、Apple ID 等第三方账号登录，我们会获取您授权的第三方账号信息（如昵称、头像等）

（2）照片信息

当您使用照片相关功能时，我们会处理：

  • 照片内容：您上传或选择的照片文件
  • 照片元数据：包括拍摄时间、地理位置信息（如果照片包含）、文件大小等
  • 人脸特征信息：用于人物识别和分类功能（敏感个人信息）
  • 照片分析结果：包括照片分类、质量评分、场景识别结果等

重要提示：人脸特征信息属于敏感个人信息，我们仅在获得您的明确同意后收集和使用，且仅用于提供人物识别和分类功能。您有权随时撤回同意。

（3）设备信息

为了提供和优化服务，我们会收集：

  • 设备型号：手机品牌、型号、操作系统版本
  • 设备标识符：设备 IMEI、IDFA（iOS）、Android ID 等唯一设备标识
  • 网络信息：IP 地址、网络类型（Wi-Fi/4G/5G）、运营商信息
  • 相框设备信息：相框设备 ID、设备名称、绑定时间、在线状态等

（4）日志信息

当您使用我们的服务时，我们会自动收集：

  • 操作日志：功能使用记录、点击行为、页面浏览记录
  • 错误日志：应用崩溃记录、错误报告
  • 性能数据：应用启动时间、页面加载时间等

（5）支付信息

当您购买订阅服务时，我们会收集：

  • 订单信息：订单号、购买时间、购买内容、支付金额
  • 支付信息：支付方式、支付状态
  • 发票信息：如果您申请发票，我们会收集发票抬头、税号等信息

我们不会收集您的支付密码、银行卡号等敏感支付信息，支付由第三方支付平台（Apple、微信、支付宝等）处理。

1.2 信息使用目的

我们收集和使用您的个人信息用于以下目的：

（1）提供核心功能

  • 账号注册、登录和身份验证
  • 照片上传、存储、管理和推送
  • 人物识别和智能分类
  • 相框设备绑定和管理
  • 订阅服务提供

（2）保障服务安全

  • 识别和防范安全威胁、欺诈行为
  • 检测和处理违法违规行为
  • 应对安全事件和用户投诉

（3）优化服务体验

  • 服务性能监控和优化
  • 用户行为分析和产品改进
  • 客户服务和用户支持

（4）数据统计分析

我们使用以下第三方统计分析服务：

国内版本：友盟+ SDK

  • 收集信息：设备信息、日志信息、使用行为
  • 使用目的：统计分析、用户画像、应用性能监控
  • 数据处理方：北京锐讯灵通科技有限公司（友盟+）
  • 隐私政策：https://www.umeng.com/page/policy

海外版本：Google Firebase Analytics

  • 收集信息：设备信息、日志信息、使用行为
  • 使用目的：统计分析、用户行为分析、应用性能监控
  • 数据处理方：Google LLC
  • 隐私政策：https://firebase.google.com/support/privacy

（5）合规要求

  • 遵守法律法规和监管要求
  • 履行与您的协议和约定
  • 应对政府部门或司法机关的要求

1.3 征得授权同意的例外

根据相关法律法规规定，以下情形中收集您的个人信息无需征得您的授权同意：

  • 与国家安全、国防安全直接相关的
  • 与公共安全、公共卫生、重大公共利益直接相关的
  • 与犯罪侦查、起诉、审判和判决执行等直接相关的
  • 出于维护您或其他个人的生命、财产等重大合法权益但又很难得到本人同意的
  • 所收集的个人信息是您自行向社会公众公开的
  • 从合法公开披露的信息中收集个人信息的，如合法的新闻报道、政府信息公开等渠道
  • 根据您的要求签订和履行合同所必需的
  • 用于维护所提供的产品或服务的安全稳定运行所必需的，如发现、处置产品或服务的故障
  • 法律法规规定的其他情形

二、我们如何使用 Cookie 和同类技术

2.1 Cookie 技术

我们或我们的第三方合作伙伴可能通过 Cookie 和其他同类技术收集和使用您的信息。

Cookie 是什么？

Cookie 是一种网络服务器存储在您设备上的小型文本文件，通常包含标识符、站点名称以及一些号码和字符。

我们如何使用 Cookie？

  • 记住您的登录状态
  • 了解您使用我们服务的方式
  • 优化广告投放效果（如果适用）
  • 安全性和反欺诈目的

您可以如何管理 Cookie？

大多数浏览器和设备允许您管理 Cookie 设置。您可以清除设备上保存的所有 Cookie，或在浏览器中设置阻止 Cookie。但如果您这么做，您可能需要在每次访问时手动更改用户设置，且部分功能可能无法正常使用。

微信 SDK 腾讯公司 设备信息、网络信息 第三方登录、分享 查看政策
支付宝 SDK 支付宝（中国）网络技术有限公司 设备信息、订单信息 支付服务 查看政策

三、我们如何共享、转让、公开披露您的个人信息

3.1 共享

我们不会与任何公司、组织和个人共享您的个人信息，除非：

（1）获得您的明确同意

在获得您的明确同意后，我们会与其他方共享您的个人信息。

（2）法定情形

根据法律法规、法律程序、诉讼或政府主管部门强制性要求。

（3）与授权合作伙伴共享

我们可能委托授权合作伙伴为您提供某些服务或代表我们履行职能，我们仅会出于本政策声明的合法、正当、必要、特定、明确的目的共享您的信息，合作伙伴只能接触到其履行职责所需的信息，且不得将此信息用于其他任何目的。

目前，我们的授权合作伙伴包括：

  • 云存储服务商：用于存储您的照片数据
  • CDN 服务商：用于加速内容分发
  • 数据统计分析服务商：友盟+（国内）、Firebase（海外）
  • 支付服务商：Apple、微信、支付宝
  • 推送服务商：用于消息推送

对我们与之共享个人信息的公司、组织和个人，我们会与其签署严格的数据保护协议，要求他们按照我们的说明、本隐私政策以及其他任何相关的保密和安全措施来处理个人信息。

3.2 转让

我们不会将您的个人信息转让给任何公司、组织和个人，除非：

  • 获得您的明确同意
  • 根据法律法规或强制性的行政或司法要求
  • 如果我们发生合并、收购或破产清算，将可能涉及到个人信息转让，我们会要求新的持有您个人信息的公司、组织继续受本政策的约束，否则我们将要求该公司、组织重新向您征求授权同意

3.3 公开披露

我们仅会在以下情况下公开披露您的个人信息：

  • 获得您的明确同意
  • 基于法律法规、法律程序、诉讼或政府主管部门强制性要求

在公开披露您的个人信息前，我们会尽可能告知您披露的目的、类型，并征得您的明确同意。

3.4 共享、转让、公开披露敏感个人信息的额外说明

对于人脸特征信息、精确位置信息等敏感个人信息，我们会采取更严格的保护措施，并在共享、转让、公开披露前单独征得您的明确同意。

四、我们如何保护您的个人信息

4.1 技术措施

我们采用业界标准的安全技术和措施保护您的个人信息：

  • 数据加密：使用 SSL/TLS 加密传输，AES-256 加密存储
  • 访问控制：严格限制数据访问权限，采用最小权限原则
  • 身份认证：多因素身份验证，防止未授权访问
  • 安全审计：定期进行安全审计和漏洞扫描
  • 防火墙：部署防火墙和入侵检测系统

4.2 管理措施

  • 数据安全管理制度：建立完善的数据安全管理制度和流程
  • 员工培训：对员工进行数据安全和隐私保护培训
  • 保密协议：与员工和合作伙伴签署保密协议
  • 安全团队：组建专业的安全团队负责数据安全

4.3 安全事件应对

尽管我们采取了上述措施，但由于技术限制和各种恶意手段，我们无法保证数据的绝对安全。

在不幸发生个人信息安全事件后，我们将：

  • 及时向您告知：安全事件的基本情况和可能的影响、我们已采取或将要采取的处置措施、您可自主防范和降低风险的建议、对您的补救措施等
  • 及时将事件相关情况以邮件、短信、电话、推送通知等方式告知您
  • 如果难以逐一告知，我们会采取合理、有效的方式发布公告
  • 同时，我们还将按照监管部门要求，上报个人信息安全事件的处置情况

4.4 数据存储

（1）存储地点

  • 中国大陆用户：您的个人信息存储在中国大陆境内的服务器
  • 海外用户：您的个人信息可能存储在您所在国家/地区或其他国家/地区的服务器

如需跨境传输您的个人信息，我们会严格遵守相关法律法规的要求，采取充分的安全保障措施，并征得您的明确同意。

（2）存储期限

我们仅在为实现目的所必需的期间内保留您的个人信息：

  • 账号信息：账号存续期间
  • 照片信息：您主动删除前或账号注销后 30 天
  • 日志信息：180 天
  • 订单信息：5 年（根据税务法规要求）

当超出上述期限后，我们会删除或匿名化处理您的个人信息。

五、您如何管理您的个人信息

我们非常重视您对个人信息的管理，并尽全力保护您的以下权利：

5.1 访问和查询

您可以通过以下方式访问和查询您的个人信息：

  • 账号信息：设置 → 个人资料
  • 相框设备：设置 → 相框管理
  • 订阅信息：设置 → 账户详情
  • 照片信息：首页 → 云端相册

5.2 更正和补充

如果您发现我们处理的您的个人信息有错误或不完整，您可以：

  • 账号信息：在个人资料页面直接修改
  • 相框信息：在相框详情页面修改
  • 其他信息：联系客服协助修改

5.3 删除

在以下情形中，您可以要求我们删除个人信息：

  • 我们违反法律法规或与您的约定收集、使用个人信息
  • 我们违反法律法规或与您的约定与第三方共享或转让您的个人信息
  • 我们违反法律法规或与您的约定公开披露您的个人信息
  • 您不再使用我们的产品或服务，或您注销了账号
  • 我们不再为您提供产品或服务

删除方式：

  • 删除照片：在照片详情页点击删除
  • 解绑相框：在相框列表页滑动解绑或在详情页点击解绑相框
  • 删除账号：设置 → 账户管理 → 删除账号

请注意：删除操作不可撤销，删除后将无法恢复。

5.4 撤回同意

您可以通过以下方式撤回您的授权同意：

  • 相册权限：在系统设置中关闭
  • 通知权限：在系统设置中关闭
  • 人脸识别功能：在应用设置中关闭

请注意：撤回相册权限将导致无法选择和上传照片。撤回其他同意可能影响部分功能使用，但不会影响此前基于您的授权而开展的个人信息处理。

5.5 注销账号

您可以通过以下方式注销账号：

  • 应用内注销：设置 → 账户管理 → 删除账号
  • 如遇问题，可发送邮件至：haokan@levect.com

注销账号后：

  • 我们将停止为您提供产品和服务
  • 我们将删除您的个人信息或进行匿名化处理（法律法规另有规定的除外）
  • 您将无法恢复账号和数据

注销账号的条件：

  • 账号处于安全状态（无被盗风险）
  • 账号无未完成的订单或服务
  • 账号无未解决的纠纷
  • 账号已解绑所有相框设备

5.6 响应您的请求

为保障您的账号安全，您的请求可能需要提供书面请求或以其他方式证明您的身份。我们会在收到您的反馈并验证您的身份后 15 个工作日内给予答复。

对于您的合理请求，我们原则上不收取费用。但对多次重复、超出合理限度的请求，我们将收取一定成本费用。

对于以下情形，我们可能无法响应您的请求：

  • 与国家安全、国防安全直接相关的
  • 与公共安全、公共卫生、重大公共利益直接相关的
  • 与犯罪侦查、起诉、审判和执行判决等直接相关的
  • 有充分证据表明您存在主观恶意或滥用权利的
  • 响应您的请求将导致您或其他个人、组织的合法权益受到严重损害的
  • 涉及商业秘密的

六、我们如何处理未成年人的个人信息

如果您是未成年人的监护人，当您对您所监护的未成年人的个人信息有疑问时，请通过本政策公示的联系方式与我们联系。

6.3 特殊保护措施

七、本政策如何更新

7.1 更新原因

我们可能适时修订本政策内容。如该等变更会导致您在本政策项下权利的实质减损，我们将在变更生效前，通过在页面显著位置提示、向您发送电子邮件或以其他方式通知您。

7.2 重大变更的定义

以下情形视为重大变更：

  • 我们的服务模式发生重大变化
  • 个人信息处理目的、处理的个人信息类型、个人信息的使用方式等发生重大变化
  • 个人信息共享、转让或公开披露的对象发生变化
  • 您参与个人信息处理方面的权利及其行使方式发生重大变化
  • 我们的联络方式及投诉渠道发生变化
  • 个人信息安全影响评估报告表明存在高风险时
