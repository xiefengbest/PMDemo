# 文档目录（docs）

本目录集中存放设计、产品与研发相关文档，并按 **手机端（App / 原型复刻）** 与 **相框端（Pad / 一体机）** 分子文件夹。根目录另保留面向终端用户的《使用说明书》（同时描述相框与 App）。

**复刻 HTML 原型**（手机端）：请以 **`手机端/Android复刻指南.md`** 或 **`手机端/iOS复刻指南.md`** 为主入口，其余为详细参考。

---

## 手机端（`docs/手机端/`）

与 **手机 App**、**H5 原型**、**Android / iOS 复刻**、上传发送、推送、算法与性能等相关的文档均在此目录。

### 复刻入口（按平台各一份）

| 文档 | 适用 | 说明 |
|------|------|------|
| **手机端/Android复刻指南.md** | Android | 页面结构、Token、资源路径、UI 规格摘要、交互要点、自查清单；目标为完美复刻 `手机端/index.html` 的 UI 与交互。 |
| **手机端/iOS复刻指南.md** | iOS | 同上，单位 pt、SF Pro、Safe Area、UIVisualEffectView 等 iOS 实现说明。 |

**使用方式**：打开 **`手机端/index.html`** 对照交互与视觉，配合 **`手机端/assets/`**（icons、drawables、welcome）与对应平台的复刻指南即可完成复刻；需查某模块精确数值时再查同目录下《UI视觉规格文档》等。

### 详细规格与参考（节选）

| 文档 | 用途 |
|------|------|
| **手机端/UI视觉规格文档.md** | 各页面/组件精确数值（字号、间距、颜色、毛玻璃、动效），双端通用 |
| **手机端/UI设计规范.md** | Design Token、组件规范、字重映射，双端通用 |
| **手机端/背景与圆角规格.md** | 背景/圆角/边框/阴影完整清单，与 `手机端/assets/drawables/` 对应 |
| **手机端/Android实施与自查文档.md** | Android Token 速查表、切图清单、实施要点（内容已汇总进《Android复刻指南》） |
| **手机端/发送任务逻辑说明文档.md** | 发送任务状态、四层反馈 UI、重试流程 |
| **手机端/照片权限交互说明文档.md** | 照片权限与空态交互说明 |
| **手机端/好看智能相框App-交互设计PRD.md** | 产品交互 PRD |
| **手机端/需求交付模板-产品给研发.md** | 产品给研发的需求交付模板 |
| **手机端/产品-H5原型-双端研发-工作流优化方案.md** | 工作流与交付物说明 |

同目录下另有算法、Push、深链、批量上传、更新说明等专题文档，可按文件名检索。

## 产品目录结构（与文档引用基准）

```
好看智能相框/
├── 手机端/
│   ├── index.html          # 主原型（复刻的视觉与交互参考）
│   ├── archive/            # index_*.html 历史备份（如有）
│   └── assets/
│       ├── icons/
│       ├── drawables/
│       └── welcome/
└── docs/
    ├── README.md           # 本文件
    ├── 好看智能相框 — 使用说明书.md
    ├── 手机端/             # App / 原型 / 双端研发文档
    └── 相框端/             # 相框（Pad）端文档
```

引用时路径以 **`好看智能相框/`** 为基准，例如：`docs/手机端/Android复刻指南.md`、`手机端/assets/icons/`。
