# 2026-03-11 更新说明

本文档整理今日原型修改内容，供 Android 和 iOS 研发同步。

---

## 一、相框列表页优化

### 1.1 无照片状态显示

**场景**：相框已绑定但尚未发送任何照片

**修改前**：显示空封面图、0张照片、空的缩略图区域

**修改后**：
- 封面区域显示相框占位图标（见 `assets/icons/icon-frame-placeholder.svg`）
- 不显示照片数量
- 不显示底部照片缩略图区域
- 仍支持点击进入详情页

**示意**：
```
┌─────────────────────────────┐
│  [相框图标]  好看智能相框    │
│             📍 客厅      >  │
└─────────────────────────────┘
```

## 二、相框详情页优化

### 2.1 无照片状态

**修改后**：
- 隐藏"全部照片"标题和分组切换器（年/月/日）
- 显示空状态引导：
  - 图标 + "还没有照片"
  - 说明文案："在「照片」页面选择照片发送到此相框"
  - 按钮："去选择照片"（点击跳转到照片页）

### 2.2 顶部导航栏变更

**修改前**：右上角显示红色「解绑」按钮

**修改后**：右上角显示「更多」图标（三个点），点击弹出操作菜单

### 2.3 相框设置菜单

点击右上角「更多」图标，弹出底部操作菜单：

| 菜单项 | 功能说明 |
|--------|---------|
| 重命名相框 | 弹出输入框修改相框名称 |
| 修改位置 | 弹出位置选择列表（见 2.4） |
| 重置人物相册 | 恢复所有被隐藏的人物及合并状态（见 2.5） |
| 解绑相框 | 原有解绑功能 |

### 2.4 修改位置

**交互**：点击后弹出位置选择列表，不支持自定义输入

**位置选项**：
- 客厅
- 书房
- 主卧
- 次卧
- 奶奶家
- 姥姥家
- 办公室
- 其他

当前选中的位置高亮显示并带勾选标记。

### 2.5 重置人物相册

**交互流程**：
1. 点击「重置人物相册」
2. 弹出确认弹窗：
   - 标题：重置人物相册
   - 描述：重置后，所有已隐藏的人物将重新显示，手动合并的人物也将恢复为初始状态。此操作无法撤销。
   - 按钮：重置
3. 确认后恢复所有隐藏的人物及合并状态
4. Toast：已重置人物相册

### 2.6 人物区域变更

**修改前**：人物头像右上角有删除按钮（×）

**修改后**：移除删除按钮，点击人物头像仅进入人物详情页

### 2.7 订阅壁纸 - 订阅更多

**修改**：点击「订阅更多」后，先关闭相框详情页，再跳转到商城页

## 三、人物详情页（从相框进入）

### 3.1 删除功能

**修改前**：删除弹窗只有一个选项"从所有相框中删除"

**修改后**：与相框详情页删除逻辑一致，提供两个选项：
- 仅从当前相框删除
- 从所有相框中删除

### 3.2 隐藏人物功能

**入口**：人物详情页 → 右上角更多 → 隐藏此人物

**交互**：
1. 点击「隐藏此人物」
2. 关闭人物详情页
3. 人物从相框人物列表中移除（移至隐藏列表）
4. Toast：已隐藏「人物名」

**恢复方式**：相框设置 → 重置人物相册

---

## 四、大图详情页删除逻辑

### 4.1 逻辑区分

| 入口来源 | 照片类型 | 删除弹窗 |
|---------|---------|---------|
| 首页照片库 | 本地照片 | 单选项「删除」+ 复选框「同时删除相机中的原始照片」 |
| 首页人物/相册 | 本地照片 | 同上 |
| 相框详情页照片 | 云端照片 | 双选项：「仅从当前相框删除」/「从所有相框中删除」 |
| 相框详情页人物 | 云端照片 | 同上 |

## 五、术语统一

### 5.1 规范

| 术语 | 适用场景 |
|------|---------|
| **删除** | 所有照片相关操作 |
| **移除订阅** / **取消订阅** | 订阅壁纸服务 |
| **解绑** | 相框设备 |

### 5.2 修改点

| 位置 | 修改前 | 修改后 |
|------|--------|--------|
| 待发送列表删除描述 | 从待发送列表中移除 | 从待发送列表中删除 |
| 解绑相框描述 | 照片将被删除 | 您发布的照片将被移除，其他用户不受影响 |
| 订阅移除完成提示 | 已从当前相框移除「xxx」 | 已取消订阅「xxx」 |
