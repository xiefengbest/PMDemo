# 好看智能相框 App — Android 复刻指南

> **目标**：完美复刻 **`手机端/index.html`**（相对产品目录 `好看智能相框/`）的 UI 与交互，供 Android 原生实现时作为唯一复刻依据。  
> **设计基准**：逻辑宽度 390dp（与 iPhone 14 Pro 一致），按密度等比适配。  
> 本文档已汇总 `UI设计规范`、`UI视觉规格文档`、`Android实施与自查文档`、`背景与圆角规格`、`发送任务逻辑说明` 等，复刻时以本文档 + **`手机端/index.html`** + **`手机端/assets/`** 为准即可。

## 二、页面与路由

### 2.1 根结构

```
App
├── 欢迎页（未绑定相框时全屏）
├── 主容器（max-width ≈ 448dp 居中）
│   ├── 顶部导航栏
│   ├── 主内容区（Tab 切换）
│   │   ├── Tab1 照片页（精选 + 人物 + 地点 + 分组照片网格）
│   │   ├── Tab2 相框页（设备列表）
│   │   ├── Tab3 商城（占位）
│   │   └── Tab4 我的（占位）
│   ├── 任务指示条（有发送任务时显示，位于 Tab 栏上方）
│   └── 底部 Tab 栏（4 Tab）
└── 全局浮层
    ├── 图片查看器（全屏）
    ├── 相册详情（右滑全屏）
    ├── 设备详情（右滑全屏）
    ├── 相框选择器（底部 Sheet）
    ├── 删除确认弹窗（底部弹出）
    ├── 任务面板（底部 Sheet）
    ├── 结果 Alert（iOS 风格居中弹窗）
    └── Toast / Banner 通知
```

### 2.2 入口与说明

| 页面/模块 | 入口 | 说明 |
|-----------|------|------|
| 照片页 | Tab「照片」 | 精选、人物、地点横向滚动 + 年/月/日分组照片网格 |
| 相册详情 | 点击精选/人物/地点卡片 | 右滑全屏，头部「全选」+ 9 宫格，底部选择栏 |
| 相框页 | Tab「相框」 | 设备卡片列表 |
| 设备详情 | 点击设备卡片 | 右滑全屏：订阅壁纸、人物、全部照片（分组+网格） |
| 图片查看器 | 点击任意照片 | 全屏黑底，返回/计数，删除/发送，左右翻页 |
| 相框选择器 | 选择栏/相册/查看器「发送到相框」 | 底部 Sheet，选相框 + 确认发送 |
| 底部选择栏 | 照片页/相册/设备详情中勾选照片后 | 取消 + 删除 + 发送到相框(N) |
| 删除弹窗 | 选择栏删除 / 查看器删除 | 底部弹出；相框内照片需「从全部相框删除」「仅从当前相框删除」 |
| 任务指示条 | 有发送任务时 | Tab 栏上方，可点击展开任务面板 |
| 任务面板 | 点击任务指示条 | 底部 Sheet，任务列表 + 进度 + 重试 |
| 欢迎页 | 未绑定相框时 | 全屏：Logo + 标题 + 描述 + 插图 + 扫码绑定 + 跳过 |

### 2.3 层级（z-index 参考）

10 导航栏 → 15–16 时间条/气泡 → 20 Tab 栏 → 40 设备详情 → 45 相册详情 → 55 查看器 → 58–60 相框选择器/删除弹窗 → 200–201 任务面板 → 210 Alert → 260+ Banner/Toast。

## 三、Design Token（直接可用）

### 3.1 颜色（colors.xml）

- **品牌/功能**：`#007AFF` 主色、`#0A84FF` 暗色主色、`#5AC8FA` 渐变末、`#3B82F6` 选中蓝、`#34C759` 成功、`#FF9500` 警告、`#FF3B30` 错误。
- **文字**：`#000000` 标题、`#1D1D1F` 正文、`rgba(60,60,67,0.6)` 二级、`#9CA3AF` 未选/禁用。
- **背景**：`#FFFFFF`、`#F3F4F6`、`#F9FAFB`、`rgba(28,28,30,0.9)` 选择栏、`rgba(0,0,0,0.3)` 遮罩、`rgba(0,0,0,0.85)` Toast。
- **边框/线**：`rgba(60,60,67,0.12)` 分隔线、`#F3F4F6` 卡片边框。

完整色值见 `docs/手机端/Android实施与自查文档.md` 第三节或 `手机端/assets/drawables/README.md`。

### 3.2 间距与圆角（dimens.xml）

- **间距**：2/4/6/8/10/12/16/20/24/32 dp（space_1～space_16）。
- **圆角**：1.5dp 进度条、6dp 分组选中按钮、8dp 分组容器/骨架、12dp 按钮/选中图、14dp Alert、16dp 卡片/面板/选择栏、20dp Banner、全圆角 9999dp 胶囊/Toast/角标。

### 3.3 字号与字重

- 20sp Bold 页面大标题；18sp Bold 导航/区域标题；17sp SemiBold Alert/面板标题；16sp Bold 子页标题；14sp Medium/Bold 列表/正文；13sp Medium 指示条；12sp 描述/按钮/百分比；11sp 副标题/胶囊；10sp 分组按钮；9sp Tab/角标；8sp 刻度。

### 3.4 毛玻璃（Blur）

- 导航/Tab 栏：blur 12dp，背景白 80%–90%。
- 底部选择栏：blur 20dp，背景 `rgba(28,28,30,0.9)`。
- 任务面板/Alert/Banner：blur 40dp + saturate(180%)，背景白 96%–97% 或对应深色。
- 未选中圆圈：blur 4dp。

### 3.5 动效曲线

- 标准：cubic-bezier(0.4, 0, 0.2, 1)；选中缩放：(0.25, 0.46, 0.45, 0.94)；圆圈弹出：(0.175, 0.885, 0.32, 1.275)；Banner/通知：(0.34, 1.56, 0.64, 1)；Alert：(0.23, 1, 0.32, 1)。

## 五、按模块的 UI 规格摘要

- **底部 Tab 栏**：4 Tab 等分，图标 22×22，标签 9sp Bold，选中 #2563EB、未选 #9CA3AF，背景白 80% + blur，内边距 16/8/24 dp（含安全区），顶边 1dp #F3F4F6。
- **顶部导航**：标题 18sp Bold 居中，右侧扫码 22×22 蓝，背景白 90% + blur，底边 1dp #F9FAFB。
- **照片网格**：年 4 列/月日 3 列，间距 2dp，1:1 单元格，占位 #F3F4F6；选中：scale 0.88、圆角 12dp、边框 3dp #3B82F6、蒙层 8% 蓝；右上圆圈 24×24、offset 6dp，未选半透明黑+blur、选中蓝+白边+阴影；勾选热区 44×44。
- **分组标题栏**：左标题 16sp Black、计数 11sp gray；右「发送至相框」10sp Bold 蓝、背景 blue_50、全圆角；右「全选」10sp Bold gray_500、背景 gray_100、全圆角；sticky 白底。
- **底部选择栏**：容器圆角 16dp、背景 rgba(28,28,30,0.9)+blur 20dp；取消 44×44 白 10%；删除 flex 1 白 5% 红图标；发送 flex 2.5 #2563EB 白字；按钮高 44dp、圆角 12dp。
- **相框选择器**：遮罩 40% 黑，面板白顶圆角 16dp；标题 16sp Bold；列表最大高 256dp；确认按钮全宽 44dp 圆角 12dp #2563EB；列表项左 48×48 封面 12dp 圆角、右 24×24 选中圈。
- **相册/设备详情**：右滑全屏，头部返回+标题+副标题，相册头右「全选」胶囊（无图标），设备头右「解绑」红胶囊；内容区同照片网格；设备删除弹窗需「从全部相框删除」「仅从当前相框删除」。
- **图片查看器**：全屏黑底，顶返回+计数 12sp 白 70%，底删除(flex 1)+发送(flex 2) 高 48dp 圆角 16dp，左右翻页 40×40 圆白 10%+blur。
- **删除弹窗**：底部弹出，标题 16sp Bold、描述 12sp gray_500，多按钮 14sp、危险项 red_500/red_50。
- **任务指示条**：Tab 上方，背景 rgba(0,122,255,0.06)，顶线 0.5dp；发送图标+文案 13sp+进度条+百分比 12sp+箭头；进度条高 3dp，填充 #007AFF（有失败 #FF9500）。
- **任务面板**：底部 Sheet，白 97%+blur 40dp，顶圆角 16dp；头部 17sp SemiBold；任务行 34×34 渐变图标、14sp 标题、11sp 副标题、进度条、重试。
- **Toast**：顶 40dp 居中，背景 85% 黑，全圆角，12sp Medium 白。
- **结果 Alert**：宽 270dp，圆角 14dp，白 96%+blur；标题 17sp SemiBold，描述 13sp；按钮 44dp、17sp #007AFF。
- **欢迎页**：Logo 区 88×88 圆角 22dp 渐变 #007AFF→#5AC8FA；标题 24sp 800；描述 15sp；插图 200×160；主按钮高 50dp 圆角 14dp；跳过 15sp 500。

更细数值见 `docs/手机端/UI视觉规格文档.md`。

## 六、交互要点（与 HTML 一致）

### 6.1 发送流程

1. 用户选择照片 → 选相框 → 确认发送。
2. 被选照片标记「发送中」（蒙层+动画），清空多选，创建任务（sending）。
3. 可弹出「开始发送」Banner；任务指示条显示进度；全部完成后根据成功/失败数得到 completed / partial / failed，弹出完成 Banner，8 秒后清理已完成任务。
4. 照片状态：sending（发送中蒙层）→ 成功 sent（已发送角标）、失败则清除 sending 可重选。

### 6.2 重试

- 入口：任务面板行内「重试」或结果弹窗「重新发送失败照片」。
- 行为：用失败照片列表新建 sending 任务，标记发送中，立即发送。

### 6.3 多选与全选

- 照片页/相册/设备详情：勾选后显示底部选择栏；相册头部「全选」为全选/取消全选当前可见照片。
- 选择栏：取消清空选择；删除弹出删除确认（相框内照片需「从全部相框删除」「仅从当前相框删除」）；发送到相框打开相框选择器。

### 6.4 相册/设备详情

- 相册：右滑进入，头部全选胶囊切换全选/取消全选，底部栏与照片页一致。
- 设备详情：订阅/人物横向滚动，全部照片为年/月/日分组+网格；删除照片弹窗必须提供「从全部相框删除」「仅从当前相框删除」。

### 6.5 图片查看器

- 点击照片进入；左右滑动或按钮翻页；底部删除/发送到相框；删除时弹窗同「从全部相框删除」「仅从当前相框删除」（设备上下文时）。

完整任务状态与四层反馈 UI 见 `docs/手机端/发送任务逻辑说明文档.md`。

## 七、自查清单

复刻完成后请逐项核对。

- **全局**：390dp 基准、安全区、字体与字重、主色/选中/错误 Token、毛玻璃使用正确。
- **Tab 栏**：4 Tab、图标 22×22、标签 9sp Bold、选中/未选色、blur、安全区、点击反馈。
- **照片页**：精选/人物/地点横向滚动；年/月/日切换器与网格 3/4 列、2dp 间距、选中态与圆圈、勾选热区 44×44、分组栏胶囊。
- **相册详情**：右滑全屏、全选胶囊、9 宫格与选择栏一致。
- **相框页与设备详情**：设备卡片样式、详情右滑、订阅/人物/全部照片、删除两选项。
- **图片查看器**：全屏黑底、返回/计数、删除/发送布局、翻页按钮。
- **相框选择器**：底部 Sheet、列表项与选中圈、确认按钮禁用态与文案。
- **删除弹窗**：底部弹出、多按钮、危险项与分隔线。
- **任务与通知**：指示条展开/收起、进度与失败色、任务面板、Toast、结果 Alert 尺寸与样式。
- **欢迎页**：全屏、Logo/标题/描述/插图/主按钮/跳过，资源来自 `手机端/assets/welcome`。
- **动效**：选中缩放与圆圈弹出曲线、面板/Sheet 滑入、按钮反馈、骨架屏（若用）。
