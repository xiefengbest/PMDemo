# 文档目录（docs）

本目录集中存放设计、产品与研发相关文档。**复刻 HTML 原型时，请以「复刻指南」为主入口，其余为详细参考。**

---

## 复刻入口（按平台各一份）

| 文档 | 适用 | 说明 |
|------|------|------|
| **Android复刻指南.md** | Android | 一份汇总：页面结构、Token、资源路径、UI 规格摘要、交互要点、自查清单。目标为完美复刻 `index.html` 的 UI 与交互。 |
| **iOS复刻指南.md** | iOS | 同上，单位 pt、SF Pro、Safe Area、UIVisualEffectView 等 iOS 实现说明。 |

**使用方式**：打开项目根目录 **`index.html`** 对照交互与视觉，配合 **`assets/`**（icons、drawables、welcome）与对应平台的 **复刻指南** 即可完成复刻；需查某模块精确数值时再查下方「详细规格与参考」。

---

## 项目根目录结构

```
好看智能相框APP/
├── index.html          # 主原型（复刻的视觉与交互参考）
├── docs/               # 本目录
├── archive/            # index_*.html 历史备份
└── assets/
    ├── icons/          # 图标 SVG
    ├── drawables/      # 背景/圆角（Android drawable XML；iOS 按 README 用代码实现）
    └── welcome/        # 欢迎页 SVG
```

引用时路径以**项目根**为基准，例如：`docs/Android复刻指南.md`、`assets/icons/`。
