# 背景与圆角规格（完整清单）

> 与 `assets/drawables/` 中 XML 及《UI设计规范》对应，便于查漏与还原。

## 一、圆角（Border Radius）

| 数值 | 单位 | 使用场景 |
|------|------|----------|
| 1.5 | dp | 进度条轨道与填充 |
| 6 | dp | 分组切换器内「年/月/日」选中按钮 |
| 8 | dp | 分组切换器容器、骨架屏矩形、照片预览条小图 |
| 9 | dp | 任务行左侧图标容器 |
| 10 | dp | Banner 应用图标、时间气泡、订阅封面文字区 |
| 12 | dp | 主操作按钮、选择栏三按钮、照片选中后图片圆角、订阅卡片、设备封面、相框列表封面 |
| 14 | dp | iOS 风格 Alert 卡片 |
| 16 | dp | 设备卡片、选择栏容器、Bottom Sheet 顶角、Action Sheet、精选卡片封面、相册/设备详情头部区域 |
| 20 | dp | Banner 通知卡片、欢迎页插图区 |
| 22 | dp | 欢迎页 Logo 容器（88×88 的圆角） |
| 全圆角 | 9999dp | 全选/取消全选、发送至相框（胶囊）、添加设备、解绑、Toast、角标「待发送」、头像、相框选择器选中圈 |

## 三、与 drawable 文件对应关系

- 所有「背景」均可与 `assets/drawables/` 下同名或语义对应的 XML 一起使用。
- 圆角数值与 `README.md` 中「圆角系统」及 `dimens.xml` 中的 `radius_*` 一致。
- 需 blur 的场景以《UI视觉规格文档》和 `assets/drawables/README.md` 中「毛玻璃」一节为准。
