# 删除与移除交互逻辑说明

## 术语规范

| 术语 | 适用场景 | 含义 |
|------|---------|------|
| **删除** | 所有照片相关操作 | 永久删除数据，不可恢复 |
| **移除订阅 / 取消订阅** | 订阅壁纸服务 | 解除订阅关系，可重新订阅 |
| **解绑** | 相框设备 | 断开设备连接，相框中照片会被删除 |

---

## 一、本地照片删除

### 1.1 首页照片库 - 批量删除

**入口**：首页选中照片 → 底部操作栏「删除」按钮

**弹窗内容**：
- 标题：`删除 X 张照片`
- 描述：`照片将从待发送列表中删除`
- 选项：`确认删除`
- 复选框：`同时删除相机中的原始照片`

**操作结果**：
- 从待发送列表中删除选中照片
- 若勾选复选框，同时删除相机中的原始文件
- Toast：`已删除 X 张照片` 或 `已删除 X 张照片，相机照片已同步删除`

### 1.2 首页照片库 - 大图详情页删除

**入口**：首页照片 → 大图详情页 → 底部「删除」按钮

**弹窗内容**：
- 标题：`删除照片`
- 描述：`确定要删除这张照片吗？`
- 选项：`删除`
- 复选框：`同时删除相机中的原始照片`

**操作结果**：同上

### 1.3 本地人物/相册详情页 - 批量删除

**入口**：首页人物/相册 → 详情页选中照片 → 底部「删除」按钮

**弹窗内容**：
- 标题：`删除照片`
- 描述：`确定要删除选中的 X 张照片吗？`
- 选项：`删除`
- 复选框：`同时删除相机中的原始照片`

### 1.4 本地人物/相册详情页 - 大图详情页删除

**入口**：首页人物/相册 → 照片大图 → 底部「删除」按钮

**弹窗内容**：同 1.3

## 二、相框照片删除（云端照片）

### 2.1 相框详情页 - 批量删除

**入口**：相框详情页选中照片 → 底部操作栏「删除」按钮

**弹窗内容**：
- 标题：`删除 X 张照片`
- 描述：`请选择删除方式，删除后照片将无法恢复`
- 选项：
  - `仅从当前相框删除`（普通样式）
  - `从所有相框中删除`（加粗样式）

**操作结果**：
- 仅从当前相框删除：只删除当前相框中的照片，其他相框不受影响
- 从所有相框中删除：删除所有相框中的该照片
- Toast：`已从当前相框删除 X 张照片` 或 `已从所有相框中删除 X 张照片`

### 2.2 相框详情页 - 大图详情页删除

**入口**：相框详情页 → 照片大图 → 底部「删除」按钮

**弹窗内容**：
- 标题：`删除相框照片`
- 描述：`确定要删除这张照片吗？`
- 选项：
  - `仅从当前相框删除`
  - `从所有相框中删除`

### 2.3 相框人物详情页 - 批量删除

**入口**：相框详情页 → 人物 → 人物详情页选中照片 → 底部「删除」按钮

**弹窗内容**：同 2.1

### 2.4 相框人物详情页 - 大图详情页删除

**入口**：相框详情页 → 人物 → 人物详情页 → 照片大图 → 底部「删除」按钮

**弹窗内容**：同 2.2

## 四、移除订阅

**入口**：相框详情页 → 订阅壁纸区域 → 点击订阅卡片右上角「×」按钮

**弹窗内容**：
- 标题：`移除订阅`
- 描述：`确定要从当前相框移除「订阅名」吗？移除后该壁纸将不再每日更新到此相框。`
- 选项：`移除订阅`

**操作结果**：
- 取消当前相框对该壁纸的订阅
- 已下载的壁纸照片保留在相框中
- 不再接收该壁纸的每日更新
- Toast：`已取消订阅「订阅名」`

---

## 五、解绑相框

**入口**：相框详情页 → 顶部「解绑」按钮

**弹窗内容**：
- 标题：`解绑相框`
- 描述：`解绑后将断开连接，您发布到该相框的照片将被移除，其他用户发布的照片不受影响。`
- 选项：`确认解绑`

**操作结果**：
- 断开与相框设备的连接
- 移除您发布到该相框的所有照片
- 其他用户发布的照片不受影响
- 相框从设备列表中移除
- Toast：`已解绑相框`

## 设计原则

1. **术语统一**：照片操作统一使用「删除」，订阅服务使用「移除订阅/取消订阅」
2. **操作确认**：所有删除操作需弹窗二次确认，防止误操作
3. **信息透明**：弹窗描述清晰说明操作影响范围
4. **可选复选框**：本地照片删除提供「同时删除相机原始照片」选项，给用户更多控制权
5. **操作反馈**：删除完成后显示 Toast 提示，明确告知操作结果
