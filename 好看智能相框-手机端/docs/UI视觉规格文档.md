# 好看智能相框 App — UI 视觉与交互还原规格

> 面向 Android / iOS 原生开发。所有数值均从 HTML 原型精确提取，请严格按照本文档还原。
>
> **设计基准**：iPhone（逻辑宽度 390pt），Android 按等比适配。

---

## 一、全局基础

### 字体

| 平台 | 字体 |
|---|---|
| iOS | SF Pro（系统默认） |
| Android | Noto Sans SC / 系统默认中文字体 |

### 常用字重映射

| HTML font-weight | iOS | Android |
|---|---|---|
| 400 | Regular | Regular |
| 500 | Medium | Medium |
| 600 | Semibold | SemiBold |
| 700 / bold | Bold | Bold |
| 800 | Heavy | ExtraBold |
| 900 / black | Black | Black |

### 颜色系统

| 语义 | 色值 | 用途 |
|---|---|---|
| 主色 / 链接 | `#007AFF` | 按钮、指示条、可点击文字 |

### 安全区

- 底部使用 `env(safe-area-inset-bottom)` 适配刘海屏 / 全面屏

### 毛玻璃效果

本项目大量使用 iOS 风格毛玻璃，原生需要对应实现：

| 场景 | blur | 附加 |
|---|---|---|
| 导航栏 / Tab 栏 | 12px (backdrop-blur-md) | 背景白色 80% 不透明 |
| 底部选择栏 | 20px (backdrop-blur-xl) | 背景 `rgba(28,28,30,0.9)` |
| 任务面板 | 40px + saturate(180%) | 背景白色 97% 不透明 |
| Banner 通知 | 40px + saturate(180%) | 背景 `rgba(30,30,34,0.95)` |
| iOS Alert | 40px + saturate(180%) | 背景白色 96% 不透明 |
