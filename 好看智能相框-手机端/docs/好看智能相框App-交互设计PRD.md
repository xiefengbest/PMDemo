# 好看智能相框 App — 交互设计 PRD

---

## 一、产品概述

好看智能相框 App 是配合"好看智能相框"硬件设备使用的手机端管理应用。用户通过 App 管理手机中待发送的照片，将照片发送到已绑定的智能相框中展示，同时支持管理多台相框设备及订阅壁纸服务。

---

## 二、信息架构

```
App
├── 照片（主Tab）—— 待发送照片
│   ├── 精选（AI分析）
│   ├── 人物（AI识别）
│   ├── 地点（AI识别）
│   └── 所有未发送（年/月/日视图）
├── 相框
│   └── 我的相框（设备列表）
│       └── 相框详情
│           ├── 订阅壁纸

│           └── 全部照片（年/月/日视图）
├── 商城（预留）
└── 我的（预留）
```

## 四、照片模块（待发送照片）

页面标题：**待发送照片**

显示手机中尚未发送到相框的照片，包含以下板块：

### 4.1 AI 智能分析

进入页面时，自动触发 AI 分析流程：

1. **初始状态**：各板块显示骨架屏 + "分析中..."/"识别中..." 提示
2. **逐步揭示**：按「精选 → 人物 → 地点」顺序，每个板块内的卡片逐个出现，当前出现的卡片右侧展示 loading 动画
3. **分析完成**：所有卡片展示完毕，loading 和提示消失

### 4.2 精选

- **横向滚动**卡片列表
- 每张卡片展示：月份名称 + 精选照片数 + 封面图（支持 1/2/3/4 张照片的不同布局）
- 含待发送照片的月份右上角显示红色**"待发送"**角标
- 点击进入**合辑详情页**

### 4.3 人物

- **横向滚动**圆形头像列表，展示姓名 + 照片数
- 点击进入**人物详情页**
- 详情页右上角"⋯"更多菜单（仅人物类型显示），包含：
  - **修改昵称**：弹出输入框修改
  - **合并人物**：将两个人物合并
  - **隐藏此人物**：从列表中隐藏

### 4.4 地点

- **横向滚动**卡片列表，封面图上叠加地点名称和照片数
- 每张卡片右上角显示红色**"待发送"**角标
- 点击进入**合辑详情页**

### 4.5 所有未发送

- **粘性标题头**：滚动时标题和年/月/日切换器吸顶
- **年/月/日切换**：切换照片分组粒度
- 每组标题也支持粘性吸顶（位于主标题下方）
- **右侧时间滚动条**：iOS 风格的点状时间轴，支持拖拽快速跳转，拖拽时显示日期气泡

### 4.6 照片选择与操作

- **选择方式**：点击照片右上角圆形勾选框进入选择模式（始终可见，无需切换模式）
- **查看大图**：点击照片图片区域进入全屏大图查看器
- **选中后**底部弹出操作栏：
  - **取消**（×）：退出选择模式
  - **删除**：弹出确认对话框，可勾选"同时从相机胶卷删除原图"
  - **发送到相框 (N)**：弹出相框选择器

## 五、合辑详情页（通用）

精选、人物、地点点击后统一进入合辑详情页，包含：

- **顶部**：返回按钮 + 标题 + 副标题（X 张照片）
- **右上角按钮**：
  - 「一键发送到相框」— 将该合辑全部照片发送到相框（直接弹出相框选择器，无需先选中照片）
  - 「⋯」更多按钮 — 仅人物类型显示
- **照片网格**：3 列网格，支持勾选和点击查看大图
- **选中后**底部弹出操作栏：取消 / 删除 / 发送到相框

---

## 六、全屏大图查看器

从任何照片网格点击图片进入：

- **顶部**：返回按钮 + 当前位置计数（如 3/12）
- **底部**：删除按钮 + 发送到相框按钮
- **左右切换**：点击左右箭头或滑动切换
- 支持从主列表、合辑详情、相框详情三种上下文进入，操作结果回写对应数据源

## 七、相框模块

### 7.1 设备列表

- 标题"我的相框" + 已绑定设备数
- 右上角"+ 添加设备"按钮（触发扫码绑定）
- 每台设备卡片展示：
  - 封面图
  - 设备名称（统一为"好看智能相框"）
  - 位置 + 照片数量（如"客厅 · 268 张照片"）
- 点击卡片进入**相框详情**

### 7.2 相框详情页

从右侧滑入的全屏页面，包含以下板块：

#### 顶部信息
- 返回按钮 + 设备名称 + **设置**按钮（⋯ 更多）
- 位置图标 + 位置名称 + 照片总数

#### 订阅壁纸
- 横向滚动卡片列表，每张卡片展示：
  - 封面图 + 壁纸名称 + 状态标签（左上角，"更新中"绿色 / "已过期"灰色）
  - "每日更新"说明文字
  - **右上角 × 按钮**：点击弹出移除确认对话框
- 末尾显示"+ 订阅更多"入口，跳转商城
- 不支持点击进入订阅详情（仅管理订阅关系，每日更新一张壁纸）

#### 人物
- 横向滚动圆形头像列表，展示姓名 + 照片数
- 点击进入人物合辑详情页（查看该人物在此相框中的照片）
- **右上角 × 按钮**：点击弹出移除确认，移除该人物的所有照片

#### 全部照片
- 粘性标题 + 年/月/日切换器
- 照片网格，每张照片左下角叠加**发送者头像和昵称**（小尺寸）
- 选择和操作逻辑同主列表
- 删除时可选择"从当前相框移除"或"从所有相框移除"

### 7.3 相框设置菜单

点击相框详情页右上角"⋯"按钮，弹出底部 ActionSheet，包含两组卡片：

| 选项 | 说明 |
|-----|------|
| **智能推荐**（V2 新增） | 居中蓝色文字 + 内联 Toggle 开关（与其他选项统一风格），关闭时自动清除推荐数据 |
| **重命名相框** | 弹出输入框修改设备名称 |
| **修改位置** | 弹出位置选择列表（客厅/书房/卧室等） |
| **重置人物相册** | 确认后重新识别人物 |
| **解绑相框** | 红色文字，弹出解绑确认 |

### 7.4 解绑

- 点击解绑按钮，弹出确认对话框
- 提示文案："解绑后将断开与该相框的连接，相框中的照片将被移除。如需重新使用，请再次扫码绑定。"
- 确认后从设备列表移除

## 八、发送到相框流程

所有"发送到相框"操作统一流程：

1. 触发发送 → 底部弹出**相框选择器**
2. 相框列表展示所有已绑定设备（封面图 + 名称 + 位置 + 照片数），支持多选
3. 底部确认按钮文案：`发送 N 张到相框`
4. 确认后：显示 loading → 完成后 toast 提示 `已发送 N 张照片到相框`

> 从相框详情页发送时，列表排除当前相框。

## 九、全局交互规范

### 9.1 弹窗规范
- 确认类操作统一使用 **iOS 风格底部 ActionSheet**（圆角卡片 + 取消按钮）
- 输入类操作使用**居中弹窗**（如修改昵称）

### 9.2 Toast 提示
- 顶部居中，黑底白字胶囊形状
- 自动消失（约 2 秒）

### 9.3 页面转场
- 二级页面（合辑详情、相框详情）统一从右侧滑入，返回时向右滑出
- 弹窗从底部上滑弹出

### 9.4 选择模式
- 所有照片网格的勾选框始终可见，点击即进入选择模式
- 选中 ≥ 1 张时底部自动弹出操作栏
- 取消全部选择或点击 × 退出选择模式，操作栏隐藏

### 9.5 照片交互分区
- **右上角区域（44×44pt）**：勾选/取消勾选
- **图片主体区域**：进入全屏大图查看
- 两个区域互不干扰

## 十、数据模型概览（供后端参考）

### 照片 Photo
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| url | string | 图片地址 |
| year | string | 年份 |
| month | string | 月份（如 "2025年1月"） |
| day | string | 日期（如 "1月15日"） |
| sent | boolean | 是否已发送到相框 |
| sender | object | 发送者信息（仅相框中的照片）{name, avatar} |

### 相框设备 FrameDevice
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| name | string | 设备名称（固定"好看智能相框"） |
| location | string | 放置位置（用于区分设备） |
| photoCount | number | 照片总数 |
| photos | Photo[] | 相框中的照片列表 |
| subscriptions | Subscription[] | 已订阅的壁纸列表 |
| persons | Person[] | 相框中识别到的人物列表 |
| aiRecommend | boolean | 智能推荐是否开启（V2 新增） |
| aiConsented | boolean | 用户是否已授权 AI 推荐（V2 新增） |

### 订阅壁纸 Subscription
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| name | string | 壁纸系列名称 |
| cover | string | 封面图地址 |
| status | string | 状态：active（更新中）/ expired（已过期） |

### 人物 Person
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| name | string | 昵称（可修改） |
| avatar | string | 头像地址 |
| photoCount | number | 照片数量 |
